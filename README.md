## Install

````
yarn install && expo install && cd ios/ && pod install && cd ..
````

## Development

### Ios
````
npx expo run:ios
````

### Android
````
npx expo run:android
````

Or you could try to launch android from Android Studio.

### Metro bundler start
````
npx expo start --dev-client
````
